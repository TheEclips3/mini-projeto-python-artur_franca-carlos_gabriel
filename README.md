# Atividade avaliativa do curso de Engenharia de Software, materia: Programação: Python
## Projeto 1 - Sistema de Cadastro de Produtos (Mini controle de estoque)
Projeto tem como princapais funções  Cadastrar, Listar ,Buscar , Atualizar e Excluir produtos sendo um sistema funcional e intuitivo para um controle de estoque. Utilizando de dicionarios, conjuntos e listas temos um controle funcional e eficiente dos produtos cadastrados pelo usuario 
### Menus do sistema e suas funcionalidades: 
- **Menu principal**: 
Cadastrar produto, listar produto, buscar produto, atualizar produto e excluir produto
## Projeto 2 - Sistema de Controle de Alunos e notas
Projeto tem como princapais funções Cadastrar e Buscar aluno,Registrar notas Listar alunos e médias, Mostrar alunos aprovados e reprovados e gerar relatórios
### Menus do sistema e suas funcionalidades:
- **Menu principal**:
Cadastrar aluno, registrar notas, listar alunos e medias, buscar aluno, mostrar aprovados e reprovaos e gerar relatorios
- **Menu relatorio**:
Alunos cadastrados, medias individuais e por fim aprovados e reprovados
